# CHANGELOG

Все заметные изменения в процессе выпуска модели кредитного скоринга фиксируются в этом файле.

Формат вдохновлён Keep a Changelog, версии следуют SemVer.

## [v1.1.0] - 2026-03-10

### Added
- Добавлен новый признак `income_stability_90d`, улучшающий качество ранжирования заявок.
- Добавлены проверки доступности нового признака в staging и shadow mode.

### Changed
- Обновлён training pipeline и схема признаков для поддержки нового агрегата.
- Обновлён отчёт по метрикам и release checklist для релизов minor-версий.

## [v1.0.1] - 2026-02-18

### Fixed
- Удалён leaky-признак, использовавший информацию, недоступную на момент принятия решения.
- Исправлен пайплайн подготовки данных и переобучена модель на корректном датасете.

### Added
- Добавлена anti-leakage проверка в CI перед выпуском patch-релизов.
- Добавлен post-incident шаг с RCA и обновлением release checklist.

## [v1.0.0] - 2026-01-20

### Added
- Выпущена первая production-версия модели кредитного скоринга с выходом в виде PD.
- Зафиксированы GitHub Flow, правила PR review и базовый release process.
- Добавлены staging/shadow validation, мониторинг drift и rollback plan.
